6.092: Assignment 4

The Boston Race organizers need a new registration organization system, and it's up to you to build it. The organizers want runners to register at most once for the race and enter the runners' results once the race ends. Additionally, the organizers want to announce the fastest and slowest runners at the end of the race

Problem

We provide two classes, Runner and Race, that provide the functionality for the registration system. You must implement the missing methods to make these classes work.

Step One: Implement Runner

First we need a class to model runners. Start by creating a class called Runner. Copy and paste the skeleton below. This class defines methods to get the name of a runner, find out if s/he's registered, and assign the race result. However, the skeleton that we provide is missing the implementations of the methods. Fill in the body of the methods with the appropriate code. The main method tests the methods. When you run the program, the output should be:

Step Two: Implement Race

Next we need to build the class that will represent a race, and manage a set of participant runners. A race can only allow a maximum number of participants before runners that try to register are rejected.

Create a class called Race. Copy and paste the skeleton below. We provide a main method that creates a 4 person race, adds some runners, runs the race, and prints some statistics. However, almost all the methods and fields are missing. You will need to define and implement the missing methods. Read the main method and look at the compile errors to figure out what methods are missing.
